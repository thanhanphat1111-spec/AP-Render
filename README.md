# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1esmgxkWJLFsy3H0cK1U4yV6P-3q4Q8rk

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. To run with the Gemini API, ensure the `GEMINI_API_KEY` environment variable is set in your execution environment. The application will also work without a key when run inside Google AI Studio.
3. Run the app:
   `npm run dev`
